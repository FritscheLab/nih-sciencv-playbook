# Glossary

- **SciENcv**: NCBI tool used to generate Common Forms and certify PDFs.
- **MyNCBI**: NCBI account dashboard hosting SciENcv and My Bibliography.
- **My Bibliography**: NCBI-managed citation library used by SciENcv for Products.
- **Common Form**: Standardized federal format for biosketch and CPOS.
- **NIH Supplement**: NIH-specific narrative additions to the biosketch.
- **CPOS**: Current and Pending (Other) Support Common Form.
- **PID**: Persistent Identifier (ORCID iD for NIH).
