# Overview: what you’re producing

For NIH, the SciENcv output is effectively:

1. **Biographical Sketch Common Form** (structured data: education, positions, products)
2. **NIH Biographical Sketch Supplement** (narratives: personal statement, contributions, honors)

{: .warning }
> **No citations** are allowed inside the Personal Statement or Contributions text boxes.  
> All references must be selected under **Products** (10 total).
