# ORCID setup (the “double handshake”)

You need **two links**:

1. **ORCID ↔ eRA Commons** (NIH identity requirement)
2. **ORCID ↔ MyNCBI/SciENcv** (data import + PID display)

{: .note }
> If either link is missing, you’ll waste time doing manual entry and risk eRA validation issues.

## Steps (PI does this)

1. Create ORCID: <https://orcid.org/register>
2. In eRA Commons: **Personal Profile → ORCID ID** → connect
3. In MyNCBI: **Account Settings → Linked Accounts** → add ORCID
4. In SciENcv, confirm your ORCID appears as the **Persistent Identifier**.
